## Summary

Added per-mode duration breakdown to `dangeresque stats` per the IMPLEMENT plan from issue-30 (staged) and the INVESTIGATE artifact. Change is purely additive to `StatsSummary`; renders a new per-mode sub-block under Durations. No artifact schema change.

## Changes

### `src/stats.ts`

1. **`StatsSummary` interface** ‚Äî added `byModeDurations: Record<string, { worker: {median, p95}; total: {median, p95}; reviewRanCount: number }>` alongside existing `workerDurationsMs` / `totalDurationsMs` fields (backwards-compatible).

2. **`computeStats`** ‚Äî accumulates three per-mode maps (`workerByMode`, `totalByMode`, `reviewRanByMode`) while iterating artifacts; after the loop, builds `byModeDurations` by iterating `Object.keys(byMode)` so every mode gets an entry (modes with no timing default to `median: 0, p95: 0, reviewRanCount: 0`).

3. **`formatStats` Durations block** ‚Äî widened label column 20‚Üí24; renamed overall rows from `worker` / `total (run)` to `overall worker` / `overall total (run)`; appended per-mode sub-block (blank separator + `<MODE> worker` rows, with `<MODE> total (run)` only when `reviewRanCount > 0`). Sorted by `byMode[m].total` desc, tiebreak `localeCompare` asc ‚Äî same as the existing `By mode (success rate)` section. Whole per-mode sub-block (including separator) is skipped when `byModeDurations` is empty.

### `test/unit/stats.test.ts`

Added 6 new tests per the INVESTIGATE plan ¬ß128-148:

1. `computeStats: per-mode durations grouped by mode` ‚Äî 3 artifacts across 2 modes; asserts each mode's median/p95 reflects its own inputs.
2. `computeStats: reviewRanCount counts only executed reviews` ‚Äî covers `review: null`, `skipped: true`, `skipped: false`; only the third counts.
3. `computeStats: empty + single-artifact modes` ‚Äî single artifact ‚Üí median=p95=that value; empty input ‚Üí `byModeDurations` is `{}`.
4. `formatStats: Durations shows per-mode block with total (run) gated on review` ‚Äî INVESTIGATE (skipped) has no total row; IMPLEMENT (ran) has both.
5. `formatStats: Durations stays within 80 cols with widest mode label` ‚Äî `INVESTIGATE total (run)` (23-char label) + 9-digit ms values; all lines ‚â§80.
6. `formatStats: per-mode block omitted when no duration data` ‚Äî empty input emits no blank separator or per-mode rows.
